# tictactoe-GUI
GUI-based Python script for the classic "Tic-Tac-Toe" game.


Details:

Python Version: Python 2.7

GUI framework: Tkinter

Algorithm: minimax

Defaults: Player1 goes first with cross (X) while the player2 goes with noughts (O).

This game is for two players, X and O, who take turns marking the spaces in a 3Ã—3 grid. The player who succeeds in placing three of their marks in a horizontal, vertical, or diagonal row is the winner.There is an option for resetting the grid when the game leads to a draw.
